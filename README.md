# The WAC Site

This is the source code for the *WAC Site*.

## Dev Stack

**Vite**
**React**
**Bulma CSS**

**NOTE**:THIS IS IN WIP

### How to run dev server

First, clone the repo.

```bash
git clone https://github.com/WAC-Site/wac-site.git
```

Go to the directory.

```bash
cd wac-site
```

Install dependencies.

```bash
npm install
```

Run the Vite dev server.

```bash
npm run dev
```

Thats it!
